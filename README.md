# Untitled UI Project

### Setup instruction

1. Clone the project

```
    git clone https://github.com/Susama012/Untitled-UI.git
```

2. Move into the directory

```
    cd Untitled-UI
```

3. install dependencies

```
    npm i
```

4. run the server

```
   npm run dev
```

### Adding dependencies

```
npm install react-router-dom 

```
